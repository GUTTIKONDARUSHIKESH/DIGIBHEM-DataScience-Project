# Movie Box Office Prediction

This project predicts movie box office performance using Python and basic data analysis techniques.

## Tools Used
- Python
- Pandas
- VS Code

## Dataset
The dataset includes movie title, genre, budget, box office collection, and ratings.

## Project Files
movie_prediction.py - Python program for data analysis  
movies.csv - Movie dataset  
ratings.csv - Ratings dataset  
project.pdf - Project report

## Output
The program loads movie data, merges datasets, and calculates average rating and average box office earnings.
